# 🎮 Jogo de Adivinhação em Python

Um projeto simples em Python onde o usuário tenta adivinhar um número gerado pelo computador.
Ideal para quem está começando a aprender lógica de programação.

---

## 🚀 Como jogar

1. Certifique-se de ter o Python instalado no seu computador.
2. Baixe ou clone este repositório:

git clone https://github.com/criis01/jogo-python.git

3. Entre na pasta do projeto:

cd jogo-python

4. Execute o jogo:

python jogo.py

5. Siga as instruções na tela e tente adivinhar o número!

---

## 🧠 Tecnologias usadas

* Python 3

* Praticar lógica de programação
